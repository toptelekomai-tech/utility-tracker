# Посты для Telegram-каналов аренды

## Вариант 1 — Короткий (лучший для каналов с ограничением)

Передаёте показания счётчиков каждый месяц вручную? 

Сделал Telegram-приложение — вводишь цифры со счётчиков, оно само считает сумму за воду, электричество, газ и выдаёт PDF-отчёт.

Бесплатно, без регистрации, работает прямо в Telegram.

👉 t.me/meter_track_bot

## Вариант 2 — С историей (для каналов типа «советы арендаторам»)

Каждый месяц одно и то же: сфоткал счётчики, открыл калькулятор, посчитал, написал арендодателю...

Надоело — сделал бота. Теперь просто ввожу показания и получаю готовый PDF с разбивкой по каждой услуге.

Работает для: электричество, холодная/горячая вода, подогрев, водоотведение, интернет.
Хранит историю за 3 года. PDF отправляю арендодателю одной кнопкой.

Попробуй → t.me/meter_track_bot
